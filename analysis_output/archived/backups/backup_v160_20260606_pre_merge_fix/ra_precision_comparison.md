## 关键发现

- RA 净新词: 估算 24,353 → 精确 19,465 (虚高 +4,888, 估算高估 20.1%)
- 这是 FR/RA 的词形重叠率 (Cap. 1-35 已学词在 Cap. 36-56 重复出现)
- 全局 (FR+RA) 末值 (精确): 30,169 词形
- 估算末值 24,353 实际不含 FR baseline 10,704, 表中'差距'列已消除 baseline 差异

## 影响判断

- 估算 RA 净新词虚高 4,888 词形 (≈ 824 词族, 1 词族 ≈ 5.93 词形)
- 全局累计末值不受影响 (精确算法本就用 FR baseline)
- ROI 甜蜜点判定: 不受影响 (甜蜜点 = 2,000 词族 ≈ 11,860 词形, 在 FR Cap. 35 范围内)
- 各 RA 章节密度变化显著: Cap. 36 从 48.8% → 25.5% (FR 词汇高重叠), 真实难度比估算低
- **结论**: 旧版估算高估了 RA 难度约 20%, 实际 RA 是 FR 知识的延续, 而非全新词汇爆炸

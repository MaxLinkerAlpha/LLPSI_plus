附录: 数据来源与方法

数据源

  • FR Cap.1-35: analysis_output/chapter_stats.csv (35 行)
  • RA Cap.36-56: analysis_output/roma_aeterna_chapter_stats.csv (21 行)
  • LLPSI 词形表: data/llpsi_corpus.db 的 fr_vocab 表 (按词形存储, 累计 9,417 unique word forms)
  • 55本扩展读物分析: analysis_output/reader_vocab_stats_v4.json
  • 35段D类拉语段: analysis_output/d_segments_vocab.json
  • 路由表: analysis_output/llpsi_reader_routing.json (191 条推荐)

算法

  1. 对每本书 / 段, 计算其 unique latin word forms (去重拉语词形)
  2. 对每章 (1-56), 计算 FR 已知词累计 (chapter_known = 1..N-1 章的 word forms)
  3. 覆盖率 cov(N) = len(book_unique ∩ chapter_known) / len(book_unique)
  4. 对每本书找最早达到 20%/40%/50%/60% 阈值的章节 → 4档可读性
  5. 教学价值 best_teach_chapter = max(len(new_set ∩ book_unique) / len(new_set))

版本历史

  • v1.0 (2026-05): 首次生成, 含 1,800词族 / 3,500词族等错误数据
  • v2.0 (2026-06-06): 全面修正数据 + 加入 5 维度扩展读物路由 (191 条推荐)

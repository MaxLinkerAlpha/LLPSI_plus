项目缘起

"Quid legam hodie?" — "我今天该读什么？"
这个问题困扰着每一位 LLPSI 学习者

LLPSI（Lingua Latina Per Se Illustrata，俗称"Ørberg 教材"）是全球最广泛使用的纯拉丁语沉浸式教材。 从 FR Cap.1 的罗马家庭，到 RA Cap.56 的罗马史，56 章构成完整的古典拉丁语旅程。

但学习过程中始终存在一个问题："我现在能读懂哪本补充读物？" Cambridge 拉丁、Oxford 拉丁、Ecce Romani、Fabulae Faciles、Hobbitus、Pugio Bruti…… 20+ 本读物，数百个章节，上万段文本——传统方式（手翻、问老师、查附录）效率极低。

LLPSI+++ 给出的答案

通过已学词汇库 × 段落可读性分析 × 56 章推荐， 自动回答"我现在应该读什么"——让学习者从第一章开始，就能"读得懂"地进入古典拉丁语的真实世界。

v3.1 可读性评分 极简版

核心思路很简单：一段拉丁语课文，你学过的单词占多少比例？ 这段课文对你来说就有多容易读。

对每段拓展读物（如 Cambridge Latin Stage 1），逐章（第 1~56 章）计算可读性分数：

可读性分数(第N章) = (已学词数 + 0.5 × 未来词数 + 0.7 × 专有名词数) ÷ 总拉丁语单词数

单词类别

类别 含义 权重
✅ 已学词 这个词在 LLPSI 第 1~第 N 章里出现过 1.0
⚠️ 未来词 这个词在 LLPSI 全书里有，但第 N 章你还没学到 0.5
⭐ 专有名词 人名、地名，在本书中出现 ≥5 次 0.7

三档推荐

难度档 可读性分数门槛 最少拉丁语单词数
🟢 流畅阅读≥ 80%≥ 15 个
🟡 挑战阅读≥ 70%≥ 20 个
🔵 选读≥ 50%≥ 30 个

快速使用

Egō sum viātor.
Egō per māre nāvigō.
Egō septem īnsulās parvas vīdī.
Prīma īnsula est Lēmnos.
Lēmnos est parvā.
Lēmnos est in marī.
In Lemno, ūnus mōns est.
In Lemno, pauca arborēs sunt.
Lēmnos est sōla.
Lēmnos est pulchra.
Secunda īnsula est Naxus.
Naxus est parvā.
Naxus est in marī.
In Naxo, multī flōrēs sunt.
In Naxo, multae herbae sunt.
Naxus est pulchra.
Naxus est nota.
Tertia īnsula est Melita.
Melita est minima.
Melita est in marī.
In Melita, ūnus homō habitat.
Homō sōlus est.
Homō piscēs capit.
Melita est parvā — sed est domūs.
Quārta īnsula est Chios.
Chios est parvā.
Chios est in Graeciā.
In Chio, multī virī habitant.
Chios est nota.
Chios est pulchra.
Quīnta īnsula est Samos.
Samos est parvā.
Samos est in Graeciā.
In Samō, multī virī habitant.
Samos est pulchra.
Samos est nota.
Sextā īnsula est Dēlos.
Dēlos est parvā.
Dēlos est sacra.
In Delo, templum deī est.
Dēlos est īnsula deī.
Multī hominēs ad Delum veniunt.
Septimā īnsula est Cyprus.
Cyprus est magnā.
Cyprus est in marī.
In Cypro, multī montēs sunt.
Cyprus est terrā bonā.
Septem īnsulae — septem sorores.
Omnēs sunt in marī.
Omnēs sunt in imperiō.
Egō hās īnsulās vīdī.
Egō hās īnsulās in corde habeō.
Et quandō in terrā sum, egō dē hīs īnsulīs cōgitō.

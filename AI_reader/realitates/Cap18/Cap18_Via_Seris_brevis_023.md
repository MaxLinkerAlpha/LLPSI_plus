**Ōrientēs — Vesper.**

Sōl in caelō. Iter longum. Via sine fīne. Ad occidentem ambulō. Sōlus. Liber.

**Vicus — Nox.**

Senex in viā. "Unde venīs?" "Ab ōriente." "Quō īs?" "Ad occidentem." "Cūr?" "Quia via est."

**Idem vicus — Māne.**

Senex alter. "Domum habēs?" "Viam habeō." "Via nōn est domus. Via est inter domōs." "Fortasse," respondē. "Fortasse via est mea domus."

**Iterum via — Merīdiēs.**

Ambulō. Sōl in caelō. Via longa. Fortasse via est domus. Fortasse domus est ubi via fīnitur. Ambulō.

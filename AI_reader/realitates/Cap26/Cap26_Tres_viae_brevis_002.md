Tres viae sunt ante viātōrem. Via prima est brevis sed perīculōsa. Via secunda est longa sed tūta. Via tertia est obscūra, nēmō scit ubi dūcit. Viātor stat. "Quam viam ēligam?" Cogitat. "Via brevis est ad mortem. Via longa est ad vitam. Via obscūra est ad nūllam partem." Sed viātor nōn est stultus. Viātor est imperātor. Imperātor dīcit: "Elegī viam tertiam." Cur? "Quia nēmō ante mē viam tertiam ēmensus est. Ego sum prīmus." Viātor intrat viam obscūram. Nūllus eum revocat. Via est sua. Via est nova.
